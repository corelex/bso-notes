2. BSO Ontology - Entities

The current BSO hierarchy has 3400 entity types. The hierarchy needs to be redesigned:

  1. split entries into naturals, functionals and complexes;
  2. consider entries like "stick" to be a natural kinds but have a
    landing place in the functional sub tree;
  3. weed out weird crap, for example, cities in France have type [[French City]], which is a complex type inheriting from [[France]] and [[City]];
  4. mine complex types from corelex
    (http://www.cs.brandeis.edu/~paulb/CoreLex/corelex.html);
